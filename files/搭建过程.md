## go-zero新建项目

```shell
# 创建项目。需要下划线，不支持中划线
goctl api new realms_web3_go

cd .\realms_web3_go\
# 管理依赖
go mod tidy
```

go.mod也需要提交到仓库中

✅ 需要提交的文件

1. 核心代码文件
      mtnode.go - 主入口文件
      internal/ 目录下的所有文件：
      internal/config/config.go - 配置结构
      internal/handler/*.go - 路由和处理器
      internal/logic/*.go - 业务逻辑
      internal/svc/*.go - 服务上下文
      internal/types/*.go - 类型定义
2. 配置文件
   go.mod - Go 模块依赖声明（必须提交）
   go.sum - 依赖校验文件（必须提交）
   etc/mtnode-api.yaml - 应用配置文件
   mt_node.api - API 定义文件
3. 项目文档
   README.md - 项目说明
   LICENSE - 许可证文件
   files/搭建过程.md - 搭建文档
